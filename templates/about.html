About This App

This Flask web application allows users to search Wikipedia articles and view short summaries.

Features

  • Search Wikipedia by entering a topic
  • Display a summary and link to the full article
  • Handles disambiguation pages and random suggestions

Technologies Used

  • Python 3
  • Flask (web framework)
  • Jinja2 (templating engine)
  • Wikipedia Python API

This app was built as part of the IT@JCU Web Development module.
